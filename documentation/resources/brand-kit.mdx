A brand kit is a collection of visual and textual elements that define a brand's identity. It can include logos, colors, fonts, images, tone of voice, and templates. Use below assets to your design.


Learn more about Solayer's brand guidelines [here](https://drive.google.com/drive/folders/1M8zgoynHuX7ynXzG4JuQW50l7QA333NJ).
