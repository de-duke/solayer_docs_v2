## API Overview

The sSOL redemption rate with native SOL is supported by a oracle price feed by [Pyth](https://pyth.network).

This oracle-based data feed calculates the sSOL redemption price relative to native SOL, ensuring that external platforms and integration partners can access accurate, up-to-date pricing information.

## Base URL

All requests to the sSOL redemption rate API should be made via the [Pyth Price Feed](https://www.pyth.network/price-feeds/crypto-ssol-sol-rr). Read more on [Pyth's Docs](https://docs.pyth.network/price-feeds/use-real-time-data/solana):
